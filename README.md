# Heartland Clutter Busters Website

This repository contains the code for the **Heartland Clutter Busters** business website.

---

## 🏠 Project Overview
Heartland Clutter Busters is a judgment-free decluttering and home cleanout service.  
This website showcases the services offered, contact info, and before/after transformations.

---

## 📂 File Structure
- `index.html` – Main website page  
- `tailwindcss.css` – Tailwind CSS file for styling  
- `README.md` – This file

---

## 🚀 How to View the Website
1. Clone the repo or download it.  
2. Open `clutterclutter.html` in your browser.  
3. The website will render with Tailwind styling.

> ⚠️ Note: The README itself is **not the website**. It's just documentation.

---

## ✨ Features
- Hero section with business name and tagline  
- Services section  
- About section  
- Before/After gallery  
- Contact info section  

---

## 📌 Optional
You can deploy this website using **GitHub Pages**:
1. Go to repo → Settings → Pages  
2. Select branch: `main` and folder: `/ (root)`  
3. Save and GitHub will give you a link to your live website
